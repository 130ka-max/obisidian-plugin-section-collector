# Section Collector

Obsidian用プラグイン。特定のキーワードを含むセクションをVault全体から検索し、サイドパネルに一覧表示します。

## 機能

### 検索機能
- **キーワード検索** - Vault内のすべてのMarkdownファイルからキーワードを含むセクションを検索
- **検索モード切替** - 「見出しのみ」または「セクション全体」から選択
- **フォルダ絞り込み** - 特定のフォルダに検索範囲を限定（複数選択可、サブフォルダ含む）
- **キーワードハイライト** - 検索結果内でキーワードを黄色でハイライト表示

### 表示機能
- **サイドパネル表示** - 検索結果を右サイドパネルに一覧表示
- **更新日時順ソート** - 結果をファイルの更新日時順（新しい順）で表示
- **セクション全文表示** - 見出しと本文を全文表示
- **ファイルを開く** - 結果クリックで新しいタブにファイルを開き、該当セクションにスクロール

### エクスポート機能
- **MDファイル出力** - 検索結果をMarkdownファイルとしてVaultルートに保存
- **選択エクスポート** - チェックボックスで選択した結果のみエクスポート可能
- **ハイライト付きエクスポート** - `==keyword==` 形式でキーワードをハイライト

## インストール

### 手動インストール
1. このリポジトリをクローン
2. `npm install` で依存関係をインストール
3. `npm run build` でビルド
4. `main.js`, `manifest.json`, `styles.css` を Obsidian Vault の `.obsidian/plugins/section-collector/` にコピー
5. Obsidian を再起動し、設定 → コミュニティプラグイン で有効化

## 使い方

1. 左サイドバーの🔍アイコンをクリック、またはコマンドパレット（`Ctrl+P`）で「Open Section Collector」
2. キーワードを入力して `Enter`
3. 必要に応じてフォルダを選択して検索範囲を絞り込み
4. 結果をクリックしてファイルを開く
5. 「📥 エクスポート」ボタンで結果をMDファイルに保存

## 開発

```bash
# 依存関係のインストール
npm install

# 開発モード（ファイル変更を監視）
npm run dev

# プロダクションビルド
npm run build
```
